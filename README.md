# MAISAWIT (AI-Driven Mobile App)

## 📱 Project Overview
MAISAWIT is an AI-powered mobile application developed to identify palm oil products and detect ingredients in real-time. The app aims to improve consumer awareness and promote transparency using modern technologies such as computer vision and natural language processing (NLP).

---

## ❗ Problem Statement
- Lack of consumer awareness regarding palm oil products  
- Limited access to accurate ingredient information  
- No efficient tools to detect palm oil content in products  

---

## 💡 Solution
Developed a mobile application that allows users to:
- Scan products using barcode  
- Detect ingredients using AI (Computer Vision & NLP)  
- Identify palm oil-related content instantly  

---

## ⚙️ Technologies Used
- Flutter (Mobile Development)  
- Python (AI / Machine Learning)  
- Firebase (Backend & Database)  
- MySQL (Database Management)  
- REST API (System Integration)
